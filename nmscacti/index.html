NMS MONITORING CACTI ON AMAZON

Tanggal: 8 Mei 2024

Network Monitoring System (NMS) merupakan sebuah sistem manajemen jaringan yang melibatkan penggunaan perangkat lunak dan perangkat keras. Perangkat lunak digunakan sebagai sistem yang mengelola proses pemantauan terhadap fungsi dan kinerja jaringan yang meliputi lalu lintas saluran data (bandwidth), pada sistem yang lebih kompleks, proses monitoring ini dapat dikembangkan sampai kepada penggunaan sumber daya, seperti sistem up/down, utilisasi CPU dan memory, serta manajemen port.

Fungsi dari NMS yang sebenarnya adalah melakukan pemantauan terhadap kualitas SLA (Service Level Agreement) dari Bandwidth yang digunakan. Hasil dari pantauan tersebut biasanya dijadikan bahan dalam pengambilan keputusan oleh pihak manajemen, disisi lain digunakan oleh administrator jaringan (Network Engineer) untuk menganalisa apakah terdapat kejanggalan dalam operasional jaringan.

Pada project kali ini sayamenginstalasi sebuah NMS monitoring tools Cacti menggunakan infrastruktur cloud Amazon Web Services (AWS). Dari topologi di atas saya membuat sebuah VPC dan di dilamnya terdapat beberapa instance. Selain itu juga saya membuat dua subnet yaitu public subnet dan private subnet.

Nah mungkin ada yang bertanya “Mengapa di bikin dua subnet?”

Karena saya membuat sebuah instance database untuk penyimpanan MySQL semua web server. kalau semisal saya menyimpan database di public itu merupakan hal yang sangat fatal karena dapat diakses oleh siapapun dan rentan kebocoran data. Untuk mengantisipasi kebocoran data saya pisahkan database dan webserver. Selain itu saya juga menambahkan keamanan tambahan yaitu BastionHost. BastionHost bertindak sebagai gerbang masuknya seseorang yang ingin meremote instance yang ada di dalam VPC (titik masuk dari eksternal). jadi ketika ada yang mau meremote instance dari luar dia harus masuk dulu ke BastionHost dan harus memiliki private key.

Dalam project ini saya memonitoring beberapa instance yang ada di dalam VPC. Adapun beberapa hal yang saya monitoring diantaranya dari segi memory, uptime, CPU, Load Avarage, dll.
